<a href="https://dev.azure.com/Link-Technologies/NMMSS%20Requirements/_workitems/edit/304/" target="_blank">2.7.5 - Generate INFCIRC-207 Inventory Change Report (ICR-207)</a> -This function produces a formatted report that reflects import/export activity for US facilities.



**Use Case(s)**

- <a href="https://dev.azure.com/Link-Technologies/NMMSS%20Requirements/_workitems/edit/455/" target="_blank">UC8.1</a>  - The Nuclear Material Analyst executes an IAEA report.

**System Requirement(s)**

- <a href="https://dev.azure.com/Link-Technologies/NMMSS%20Requirements/_workitems/edit/462/" target="_blank">IAEARPT7</a> - The NMMSS shall have the ability to produce an INFCIRC/207 inventory change report that will reflect import and export activity for U.S. facilities. The report will not include import and export for U.S. facilities selected or attached. Important items that will be shown on this report are report number, entry number, action date, batch name, element weight, isotope weight, and back reference report and entry number, if applicable (ICR-207).
