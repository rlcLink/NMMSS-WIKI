This function produces a formatted report of all DOE contractors that have authorized the use of telephone conversations as a basis for correcting input documents, if errors are detected during NMMSS processing.

**Use Case(s)**

- UC1.2 - The Nuclear Material Analyst executes an Authority Reference report.

**System Requirement(s)**

- AURPT16 - The NMMSS shall be able to produce a DOE contractor authorization to change data on DOE/NRC forms reference report (D-040A-2) that will list all DOE contractors which have authorized the use of telephone conversations as a basis for correcting input 7 documents if errors are detected during NMMSS processing.
