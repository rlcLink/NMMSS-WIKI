<a href="https://dev.azure.com/Link-Technologies/NMMSS%20Requirements/_workitems/edit/293/" target="_blank">2.6.2 - Generate Log Report (SYLOGRRPT)</a> - This function produces a log report for internal use only that will compare the D-4A report with the actual output reports that have been generated for a specified batch run. Directory reports will be included.



**Use Case(s)**

<a href="https://dev.azure.com/Link-Technologies/NMMSS%20Requirements/_workitems/edit/785/" target="_blank">UC6.1</a> - The Nuclear Material Analyst executes a System report.
